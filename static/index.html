Welcome to the Yoda IM

This server provides API endpoints for searching for and accessing data from OOI CI

API Endpoints Examples

What kinds of data are available from OOI?

/products

/products?filter=`some_word`

Examples

  • All products: /products
  • Filtered example: /products?filter=sal

Key Points

  • You can use these names to find deployments that generated a particular parameter

What deployments produced the data that I want?

/deployments/find/`parameter name`

Examples

  • Deployments with Salinity: /deployments/find/salinity
  • Deployments with Wave Direction: /deployments/find/Wave Direction

Key Points

  • This is a list of deployments that contain the parameter you want.
  • Each record will have a reference designator. You need that to get the data.
  • There's also lat, lon, start datetime, and end datetime. You can filter by position and dates for the sites that you are interested in.

How do I get the data for a deployment?

/data/`reference designator`?username=`ooi username`&token=`ooi user token`&begin_date=`yyyy-MM-ddTHH:mm:ssZ`&end_date=`yyyy-MM-ddTHH:mm:ssZ`
/data/CE04OSBP-LJ01C-06-CTDBPO108?username=OOIAPI-AB06CTD1GFAGGC&token=ABCDEFGHIJKL&begin_date=2017-08-23T00:00:00.000Z&end_date=2017-08-23T00:00:00.000Z
